## 📋 Sobre el Proyecto

**NEURA-SYNC** es el diseño de una Landing Page enfocada en conversión técnica, cuyo objetivo es presentar el despliegue de agentes virtuales que resuelven tickets de soporte sin intervención humana. El diseño transmite tecnología de punta a través de micro-animaciones, "Glassmorphism", y contrastes eléctricos.

### ✨ Características Clave del Diseño

- **Cyber-Minimalism (Dark Mode)**: Fondo `#050505` con halos de luz (Glow effects) y acentos en Azul Cobalto (`#0047FF`).
- **Glassmorphism**: Componentes flotantes y tarjetas "esmeriladas" para agrupar los componentes de información visual.
- **Bento Grid System**: Grilla de características asimétricas que destacan las mayores ventajas competitivas de la plataforma de IA.
- **Data Simulation en Vivo**: Dashboard simulado animado por Vanilla JavaScript que grafica ahorro de costos (en `CLP` 🇨🇱) e ilustra el desempeño de los agentes de soporte generativo.
- **Asimetría del Tiempo**: Una comparativa visual en formato barra de estado para ejemplificar los tiempos de respuesta de IA vs Humanos.

## 🛠️ Tecnologías Empleadas

Este proyecto ha sido desarrollado utilizando tecnologías base para maximizar el rendimiento y la facilidad de mantenimiento sin depender de librerías externas pesadas.

- **HTML5**: Semántica estructurada y SEO-friendly.
- **CSS3 (Vanilla)**: 
  - Variables nativas CSS para el sistema de diseño (`--bg-dark`, `--accent-blue`, etc).
  - Animaciones Keyframe para gradientes y fondos orgánicos.
  - Flexbox & Grid para el layout responsivo (`Bento Grid`).
- **JavaScript (Vanilla)**: 
  - `IntersectionObserver` para lograr animaciones de entrada progresivas en scroll.
  - Generación de datos simulados y formateo dinámico local (Pesos Chilenos a través de `Intl.NumberFormat`).

## 🚀 Cómo Empezar (Desarrollo Local)

Ejecutar esta Landing Page en un entorno local es extremadamente sencillo, ya que no requiere ningún proceso de compilación (*Build step*).

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/MauriiLogx/Neura-sync-landing.git
   ```
2. **Navegar al directorio principal**
   ```bash
   cd Neura-sync-landing
   ```
3. **Ejecutar en el navegador**
   - Puedes abrir directamente el archivo `index.html` en cualquier navegador moderno.
   - Alternativamente, para una mejor experiencia de desarrollo utiliza extensiones como **Live Server** en VS Code o ejecuta un servidor simple a través de la terminal:
   ```bash
   # Utilizando Python 3:
   python -m http.server 8000
   ```
   > Luego visita `http://localhost:8000/` en tu explorador.
